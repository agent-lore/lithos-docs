# Deployment

Lithos is designed to run on your own infrastructure. Choose the deployment method that fits your setup.

| Method | Best for |
|--------|---------|
| [Docker](docker.md) | Most users — zero-dependency, easy to manage |
| [Self-hosted (bare metal)](self-hosted.md) | Advanced users, Raspberry Pi, VPS, custom setups |

For local development, see [Installation](../getting-started/installation.md).
